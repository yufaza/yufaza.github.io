Tentang saya

Nama saya M. Yusuf Fajar Zaelani saya lahir di Sukabumi pada hari kamis 8 oktober 1998. Lulusan Sarjana Teknik Informatika Universitas Muhammadiyah Sukabumi pada tahun 2020 dengan predikat cumlaude (IPK 3.85). Saya tertarik mencari pengalaman baru di dunia pekerjaan selain untuk menambah pengalaman saya saya juga ingin menambah relasi serta belajar bagaimana tantangan dan hambatan kehidupan di dunia kerja. Saya juga sering membangun sebuah project web baik itu untuk sebuah perusahaan, sekolah ataupun untuk individu.

Pengalaman berorganisasi saya pernah menjadi bagian kabinet himpunan mahasiswa teknik informatika di divisi Minat dan Bakat, saya juga pernah masuk sebuah komunitas gerakan literasi di kampus yang bernama library lovers community disana saya menjabat sebagai divisi sumber daya manusia dan juga pernah terlibat sebagai ketua pelaksana di salah satu kegiantannya dan ada beberapa organisasi lain yang pernah saya ikuti.
